pTicTacToe
==========

A LWJGL Java TicTacToe game, formerly C++ WinBGIm.

Potent TicTacToe (pTicTacToe) is a LWJGL/OpenGL Java program 
that allows two users to play a game of 3-by-3 TicTacToe against each other.
